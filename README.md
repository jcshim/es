* 🔰 **입문용** (간단하고 바로 작동되는 것)
* ⚙️ **중급** (센서 여러 개, 프로그래밍 약간)
* 🤖 **고급 / 창의 작품** (IoT, 로봇, 스마트홈 등)

일단 지금은 난이도별로 몇 가지 예시를 드릴게요:

---

### 🔰 **입문용 프로젝트**

1. **자동 조명 시스템**

   * 조도 센서 + LED
   * 어두워지면 자동으로 불 켜지는 시스템
2. **디지털 온도계**

   * 온도 센서 (DHT11 등) + LCD 출력
   * 현재 온·습도 표시
3. **초음파 거리 측정기**

   * 초음파 센서 + LCD
   * 앞에 있는 물체 거리 측정

---

### ⚙️ **중급 프로젝트**

1. **스마트 화분**

   * 토양 수분 센서 + 릴레이 + 워터펌프
   * 흙이 마르면 자동으로 물 주기
2. **RFID 도어락**

   * RFID 카드 리더 + 서보모터
   * 등록된 카드만 문 열림
3. **블루투스 제어 RC카**

   * 블루투스 모듈 + 모터드라이버
   * 스마트폰 앱으로 이동 제어

---

### 🤖 **고급 / 창의 프로젝트**

1. **스마트 미러 (Smart Mirror)**

   * 아두이노 + LCD + 센서 + WiFi 모듈
   * 시간, 날씨, 일정 표시
2. **IoT 실내 환경 모니터링**

   * 온도/습도/미세먼지 센서 + WiFi
   * 스마트폰 앱이나 웹으로 실시간 데이터 확인
3. **음성 인식 조명 제어**

   * 음성 인식 모듈 + LED
   * “불 켜”, “불 꺼” 명령에 반응
